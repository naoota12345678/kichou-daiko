# acc-tool 電子帳簿保存法 対応状況

## 電帳法とは

2024年1月から、**電子取引で受領した領収書・請求書はデータのまま保存が義務**。
Amazon・楽天で買い物した領収書を紙に印刷して保存するのはNG。
違反すると青色申告の取消リスクがある。**全事業者が対象。**

## このツールが満たしている要件

### 1. 検索要件（3項目で検索可能な状態で保存）

電帳法は「日付・金額・取引先で検索できること」を求めている。

**acc-toolの対応:**
- ファイル名に **日付・取引先・品名・金額** を全て含めて保存
- 例: `20250312_Amazon_東芝オーブンレンジ_48804.pdf`
- OS標準のファイル検索（エクスプローラー、Finder）で即座に検索可能
- Googleドライブにアップロードすればドライブの検索機能でも対応

> 売上高1,000万円以下の小規模事業者は、税務職員のダウンロード要求に応じられれば検索要件すら全て不要（国税庁FAQ問32）。ファイル名に含めておけばどちらにも対応。

### 2. 真実性の確保（改ざん防止）

電帳法は以下のいずれかを求めている:
- (1) タイムスタンプ付与
- (2) 訂正削除の履歴が残るシステム
- (3) 訂正削除の防止に関する事務処理規程の策定・運用
- (4) タイムスタンプ付きデータの受領

**acc-toolの対応:**
- **(3)の事務処理規程を自動生成** する機能あり（`acc gen-regulation "会社名"`）
- 国税庁の公開サンプルに準拠した規程を即座に出力
- **タイムスタンプは不要**（事務処理規程で代替可能）

### 3. 可視性の確保（見読可能性）

電帳法は「速やかに出力できること」を求めている。

**acc-toolの対応:**
- PDF/JPEG形式で保存 → PC＋ディスプレイ＋プリンタで即表示・印刷可能
- 特別なソフト不要、標準のPDFビューアで閲覧可能

### 4. 保存期間

法人: 7年間（欠損金がある場合は10年間）

**acc-toolの対応:**
- ローカル or Googleドライブに保存
- 月50件 x 12ヶ月 x 7年 = 約4.2GB → Google無料枠15GBで十分

### 5. 整理された保存体系

**acc-toolのフォルダ構成:**
```
receipts/
  2024年度/
    01月/領収書/
      20250104_Amazon_コンセントカバー_2999.pdf
      20250115_Amazon_モバイルモニター_10980.pdf
    02月/領収書/
      20250212_Amazon_はっとり海苔_1588.pdf
    03月/領収書/
      20250312_Amazon_東芝オーブンレンジ_48804.pdf
  2025年度/
    04月/領収書/
      20250411_Amazon_換気扇フィルター_3125.pdf
    ...
```

- 会計年度ごと（4月始まり）に自動分類
- 月別フォルダに自動整理
- 重複チェック機能あり（同じ領収書の二重保存を防止）

## 訴求ポイント

1. **「ボタン1つで電帳法対応が終わる」** — Amazonにログインするだけで、領収書PDF保存＋ファイル名付与＋フォルダ整理＋仕訳CSV生成が全自動
2. **「タイムスタンプ不要、事務処理規程も自動生成」** — 高額なタイムスタンプサービスは不要。国税庁準拠の規程をコマンド1つで生成
3. **「freeeでも解決できないAmazon問題を解決」** — freeeのカード明細は「AMAZON.CO.JP ¥12,500」としか出ない。品目がわからないから科目が決められない。このツールなら品名まで自動取得
4. **「データは全部自分の手元」** — 他社サーバーに預けない。自分のPC/Googleドライブに保存
5. **「月500円でプロ級の電帳法対応」** — バクラク・TOKIMUMは月数千円〜数万円。このツールは無料〜500円

## 実績（2025年データで検証済み）

- Amazon 2025年分: **47件の注文を自動取得、34件の領収書PDFを自動保存**
- 全ページ自動ページ送り（5ページ分）
- 電帳法準拠ファイル名で年度/月フォルダに自動整理
- 所要時間: 約3分（ログイン含む）
